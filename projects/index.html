RC4-drop

RC4-drop

Object-oriented implemention of RC4-drop[n], a modification of the simple and speedy RC4 stream cipher that defends against the Fluhrer, Mantin and Shamir attack by discarding the first n bytes of the keystream after key generation. It can also be used as regular RC4 by setting n = 0, or a performant pseudorandom number generator.

CTFOJ

CTFOJ

An open-source online judge specifically designed for hosting interactive cybersecurity challenges and contests. Features include practice problems, contests, an admin console, two-factor authentication, a comprehensive test suite, and live rankings. Built with Flask, in collaboration with jdabtieu.
